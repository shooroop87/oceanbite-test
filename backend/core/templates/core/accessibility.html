Accessibility Statement

Our Commitment to Accessibility

Oceanbite Seafood is committed to ensuring digital accessibility for all users, including people with disabilities. We strive to continuously improve the user experience for everyone and apply relevant accessibility standards to make our website usable by all.

Accessibility Standards

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards published by the World Wide Web Consortium (W3C). These guidelines help make web content more accessible for people with disabilities and user-friendly for everyone.

Key Accessibility Features

  • Semantic HTML: Proper HTML structure and semantic elements for screen readers
  • Keyboard Navigation: Full website functionality available via keyboard
  • Alt Text: Descriptive alternative text for all images
  • Color Contrast: High contrast ratios for text and backgrounds
  • Focus Indicators: Clear visual focus indicators for interactive elements
  • Descriptive Links: Meaningful link text that describes the destination
  • Form Labels: Clear labels and instructions for all form fields
  • Heading Structure: Logical heading hierarchy for easy navigation

Assistive Technology Support

Our website is designed to work with commonly used assistive technologies, including:

  • Screen readers (JAWS, NVDA, VoiceOver, TalkBack)
  • Voice recognition software
  • Keyboard navigation tools
  • Screen magnification software
  • High contrast and dark mode browser settings

Browser and Device Compatibility

We test our website across multiple browsers and devices to ensure compatibility:

Supported Browsers

  • Chrome (latest versions)
  • Firefox (latest versions)
  • Safari (latest versions)
  • Microsoft Edge (latest versions)

Mobile Accessibility

Our website is fully responsive and includes mobile-specific accessibility features such as touch-friendly interface elements and proper zoom functionality.

Content Accessibility

Text and Font

  • Clear, readable fonts with appropriate sizing
  • Sufficient spacing between lines and paragraphs
  • Text can be resized up to 200% without loss of functionality
  • Simple, clear language with business terminology explained

Images and Media

  • All product images include descriptive alt text
  • Decorative images are properly marked for screen readers
  • Any videos include captions when available
  • Charts and graphs include text descriptions

Navigation and Structure

  • Consistent navigation across all pages
  • Skip navigation links for keyboard users
  • Clear page titles and headings
  • Breadcrumb navigation where applicable

Accessibility Testing

We regularly test our website using:

  • Automated accessibility testing tools
  • Manual keyboard navigation testing
  • Screen reader testing
  • Color contrast verification
  • Mobile device accessibility testing
  • User feedback and testing

Known Issues and Limitations

While we strive for full accessibility, we acknowledge that some areas may need improvement:

  • Some third-party embedded content may not meet full accessibility standards
  • Complex product filtering interfaces are being enhanced for better screen reader support
  • PDF documents are being reviewed and updated for accessibility compliance

We are actively working to address these issues and continuously improve our accessibility implementation.
